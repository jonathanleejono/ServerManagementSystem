# ServerManagementSystem

This is an app where users can manage servers they can access locally. A user can ping the server to see if the server is up and monitor its status.

### Backend:

- Java, Spring Boot, Lombok, Maven
- MySQL (Database)

### Frontend:

- Angular (TypeScript)
